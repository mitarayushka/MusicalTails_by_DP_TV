# Musical Tails

**Musical Tails** — это настольная ритм-игра на WPF, в которой игроку необходимо вовремя нажимать клавиши в такт музыке.

---

## Описание

Игра построена на реакции и чувстве ритма: на экране появляются ноты (или сигналы), и задача игрока — нажимать соответствующие клавиши в нужный момент.

Чем точнее попадание — тем больше очков получает игрок.

---

## Основные возможности

* Запуск новой игровой сессии
* Нажатие клавиш в ритм музыки
* Подсчёт очков (Score) в реальном времени
* Обновление счёта с высокой частотой (каждые ~100 мс)
* Завершение сессии с сохранением результата (планируется)
* Простой и понятный интерфейс на WPF

---

## Геймплей

* На экране появляются визуальные элементы - клавиши
* Игрок должен нажимать соответствующие клавиши в нужный момент
* За точные попадания начисляются очки
* При промахах очки не начисляются, но при этом теряется жизнь (всего их 3)
* Итоговый результат зависит от реакции игрока
  
### ВАЖНО!!!:
* *Перед тем, как играть, нужно запомнить значение каждой клавиши:*
* $$ \color{green}{\text{Зелёная - один тык}} $$
* Розовая - два тыка
* Синяя - долгий один тык
* Красная - клавиша-ловушка (не нажимать!)

## Технологии

* C# 12
* .NET 8
* WPF (Windows Presentation Foundation)

---

## Структура проекта

* **Core**
  Содержит бизнес-логику игры:

  * обработка очков (Score)
  * игровые данные
  * логика ритма и попаданий

* **UI (WPF)**

  * `MainWindow.xaml` — главное меню
  * `DataWindow.xaml` — игровое окно (сессия)

---

## Как запустить проект

* Клонируйте репозиторий или откройте папку проекта
* Убедитесь, что целевая платформа — .NET 8.0 Windows (или выше)
* Откройте решение ProjectExample.sln в среде разработки (VS 2022 / Rider / VS Code, если он настроен)
* Запустите проект без отладки (с отладкой может повисать)
